О презентации

Это встраиваемая презентация, размещённая на внешнем сервисе. Здесь мы оборачиваем её в простой статический сайт для публикации через GitHub Pages.

Исходники

Файл index.html в корне репозитория содержит сайт. Для автоматического деплоя настроен workflow GitHub Actions.

Развёрнуто с помощью GitHub Pages — обновляется при пуше в ветку main (workflow создаёт ветку gh-pages).
